This repository contains a Python-based machine learning project designed to predict the Closing price od a stock using historical market data.It leverage time-series analysis
and regression techniques to forecast future price movements.

#Project Overview 
-> Prediction the stock market is a complex task due to its volatile nature. This project aims to simplify the [process by analysing historical trends ,calculating technical indicators and training a model to estimate the closing price of a stock with resonable accuracy.
**Tech Stack**
->Language: Python 3.x
->Data Analysis: Pandas and Numpy
->Machine Learnig: Scikit-learn
->Visualization: Streamlit
